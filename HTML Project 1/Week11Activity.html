Binary Search Algorithm (Recursive)

Bina ry search is an efficient algorithm to search for a target item in a sorted list.

Link to Binary Search Algorithm

A recursive version of binary search takes two additional input values, low and high. If n is the number of items in the list, then variables low and high must lie in the range 1 through n and must satisfy the inequality low ≤ high. The variables low and high denote a range of candidate locations {low, low+1, …, high} so that if x is in the list at all, then x must be reside in one of the locations in the range.

Method Steps

  1. If (low = high AND alow = x), Return(low)
  2. If (low = high AND alow ≠ x), Return(-1)
  3. mid := ⌊(low + high)/2⌋
  4. If (x ≤ a_mid), then high := mid
  5. If (x > a_mid), then low := mid + 1
  6. Return( RecBinarySearch(low, high, A, x) )
